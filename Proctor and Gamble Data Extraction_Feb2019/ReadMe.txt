Downloaded PDFs are converted into text files

Text Files are parsed for:
- Revision Date (Header)
- Product Name (Section 1)
- Product Identifier (Section 1)
- Recommended use (may not be available on all SDS) (Section 1)
- Manufacturer (Section 1, only need 1st line, do not need address)
- Chemical name (Section 3)
- Synonyms (Section 3)
- CAS-No (Section 3)
- Weight % (Section 3) (Note alongside numeric concentration we should also store a field 'units' where units = weight %) 

Python 3.6
