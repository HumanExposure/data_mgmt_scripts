Data were initially scraped directly from product webpages at https://living-future.org/declare/ using selenium.
Resulting dataframe was cleaned before export to csv.
Scraped data were written into text files
Text Files were converted to PDF Files.

Data includes:
- CAS Number
- CDPH (Yes, No)
- Company
- Component
- Declaration Status
- Declaration ID
- End of Life (Recyclable, Salvageable/Resuable in its Entirety, Landfill, etc.)
- Final Assembly Location
- Ingredient
- License Expiration
- Percent
- Source
- VOC

Python 3.6
